# SK Kahinoor — Premium 3D Developer Portfolio

A highly custom, immersive 3D developer portfolio showcasing the skills, experiences, and commercial projects of **SK Kahinoor**, Full Stack Web Developer.

Built with React, Vite, Framer Motion, and React Three Fiber (Three.js) to deliver premium animations, advanced glassmorphism visuals, and hardware-accelerated rendering.

---

## 🚀 Key Features

*   **3D Interactive Hero Canvas**: Built with React Three Fiber (R3F) and `@react-three/drei`, rendering a floating cluster of 3D programming languages (PHP, Laravel, JS, React, MySQL) that rotate dynamically in space and scale responsively based on viewport width.
*   **Bento Grid Tech Stack**: Asymmetrical visual layout separating frontend technologies, backend proficiencies, tools, and circular SVG progress indicators that animate on scroll.
*   **3D Hover Project Cards**: Feature-rich project showcases for **MyBookHub** and **FreeBazar** with full 3D interactive tilt physics, accent lighting glows, real live URLs, and custom stat badges.
*   **Responsive Timeline**: Staggered alternating left-right experience nodes that collapse into a left-aligned vertical mobile layout seamlessly using responsive CSS grids.
*   **Premium Ambient Theme**: Sleek deep blue and absolute black background with slow-drifting hardware-accelerated glowing gradient orbs.
*   **True Glassmorphism**: Cards styled with directional lighting borders (`border-top`, `border-left` highlights) and inner box-shadow edge reflections representing real glass panels.

---

## 🛠️ Technology Stack

*   **Frontend Library**: React 18+ (with Vite)
*   **Styling**: Vanilla CSS3 + Modern variables
*   **3D Elements**: Three.js, `@react-three/fiber`, `@react-three/drei`
*   **Animations**: Framer Motion (useScroll, useSpring, useTransform, useInView)
*   **Icons**: React Icons (Fa, Si)
*   **Routing & Scroll**: IntersectionObserver API + smooth anchors
